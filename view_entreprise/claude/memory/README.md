# claude/memory/

Ce dossier contient les fichiers de mémoire de Claude pour ce projet.

## Convention de nommage

```
YYYY-MM-DD-nom-descriptif.md
```

Exemples :
- `2026-04-08-contexte-initial.md`
- `2026-04-08-module-commandes-genere.md`
- `2026-04-09-correction-types-user.md`

## Règle

À chaque session de travail significative, Claude crée un fichier ici pour mémoriser :
- Ce qui a été fait
- Les décisions prises
- Les corrections appliquées
- Les points d'attention pour la prochaine session

Ces fichiers sont lus en priorité au démarrage d'une nouvelle session pour reconstituer le contexte.
