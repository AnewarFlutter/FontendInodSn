# FontendInodSn

Workspace principal du projet.

## Contenu

- `view_entreprise/`: application frontend Next.js (React, TypeScript, Tailwind).

## Demarrage rapide

1. Aller dans le projet frontend:

```bash
cd view_entreprise
```

2. Installer les dependances:

```bash
npm install
```

3. Lancer en developpement:

```bash
npm run dev
```

4. Build de verification:

```bash
npm run check
```

## Git
